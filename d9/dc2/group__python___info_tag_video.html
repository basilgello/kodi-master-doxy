Detailed Description

Kodi's video info tag class.

Class: xbmc.InfoTagVideo([offscreen])

Access and / or modify the video metadata of a ListItem.

Example:

...
tag = xbmc.Player().getVideoInfoTag()
title = tag.getTitle()
file = tag.getFile()
...
