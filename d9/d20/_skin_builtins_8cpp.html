static int SelectBool ( const std::vector< std::string > &  params)
static

Select and set a skin bool setting.

Parameters
paramsThe parameters.

params[0] = Number of a localized string to display as a header in a select dialog

params[1,...] = one or more number|skinbool-setting pairs where number is index of a localized string used as label

and skinbool-setting is a string of the skinbool setting name. The pairs are added to the select dialog list.

If the users confirms a (single) selection label in the select dialog, the paired skinbool is set to true and all others

in the list are set to false. Multi-select is not available.
