Start a slideshow.

Parameters
paramsThe parameters.

params[0] = Path to run slideshow for. params[1,..] = "recursive" to run a recursive slideshow. params[1,...] = "random" to randomize slideshow. params[1,...] = "notrandom" to not randomize slideshow. params[1,...] = "pause" to start slideshow paused. params[1,...] = "beginslide=<number>" to start at a given slide.

Set the template parameter Recursive to true to run a recursive slideshow.
