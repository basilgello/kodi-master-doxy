Handle messages on window.

Parameters
messageGUI Message that can be reacted on.
Returns
if a message can't be processed, return false

On these messages this class reacts.
When retrieving...

  • GUI_MSG_WINDOW_DEINIT
    ...the last focused control is saved to m_iLastControl.
  • GUI_MSG_WINDOW_INIT
    ...the musicdatabase is opend and the music extensions and shares are set. The last focused control is set.
  • GUI_MSG_CLICKED
    ... the base class reacts on the following controls:
    Buttons:
    • CONTROL_BTNVIEWASICONS - switch between list, thumb and with large items
    • CONTROL_BTNSEARCH - Search for items
      Other Controls:
    • The container controls
      Have the following actions in message them clicking on them.
      • ACTION_QUEUE_ITEM - add selected item to end of playlist
      • ACTION_QUEUE_ITEM_NEXT - add selected item to next pos in playlist
      • ACTION_SHOW_INFO - retrieve album info from the internet
      • ACTION_SELECT_ITEM - Item has been selected. Overwrite OnClick() to react on it
