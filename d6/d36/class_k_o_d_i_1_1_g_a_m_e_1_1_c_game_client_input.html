◆ m_joysticks

JoystickMap KODI::GAME::CGameClientInput::m_joysticks
private

Map of port address to joystick handler.

The port address is a string that identifies the adress of the port.

The joystick handler connects to joystick input of the game client.

This property is always populated with the default joystick configuration (i.e. all ports are connected to the first controller they accept).
