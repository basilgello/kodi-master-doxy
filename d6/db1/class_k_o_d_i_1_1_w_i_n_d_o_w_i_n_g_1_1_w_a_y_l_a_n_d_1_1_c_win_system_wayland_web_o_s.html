bool KODI::WINDOWING::WAYLAND::CWinSystemWaylandWebOS::SetExportedWindow ( CRect  orig,
CRect  src,
CRect  dest 
)

Sets up the an exported window for display. The display engine will merge the exported window with the UI layer. Therefore any UI element must be transparent for the exported window to punch through. Not available on non wayland-webos-foreign devices (pre webOS 5)

Parameters

srcSource window rect crop window
dstDestination rect
Returns
True on success, else false
